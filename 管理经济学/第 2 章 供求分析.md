# 供求分析

## 2.1 需求分析

1. 需求

   需求是指消费者在一定时间内在各种可能的价格水平下愿意且能够购买的某种商品或服务的数量。

2. 影响需求的因素

   （1）商品的价格。  
    <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-1.png" width="600"/>

   （2）消费者的收入水平。

   （3）相关商品的价格。

   （4）消费者的偏好。

   （5）其他诸如消费者对未来价格的预期、市场规模以及地区特殊因素等。

3. 需求函数

   需求函数可以表示为: **Qd = f(P)**

4. 需求曲线

   在正常情况下，由于需求量与价格呈反方向变动，所以需求曲线是一条向右下方倾斜的曲线，需求曲线的斜率为负。  
   <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-2.png" width="600"/>

5. 需求量的变动和需求的变动

   需求量变动随着价格的变动而变动

   需求的变动是指在某商品价格不变的条件下，由于其他因素变动所引起的该商品的需求量的变动。

## 2.2 供给分析

1. 供给

   供给是指在某一特定时期内，在每一价格水平上，生产者愿意而且能够生产的物品与劳务的数量。

2. 影响供给的因素

   1）商品的价格。  
   当一种商品的价格上升时，供给量就会增加;随着价格降低，供给量就会减少。这一规律被称为供给向上倾斜规律。  
   <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-4.png" width="600"/>

   2）生产成本。

   3）替代品的价格。

   4）生产者对价格的期望。

3. 供给函数

   供给函数可以表示为: **Qs = f(P)**

4. 供给曲线

   <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-5.png" width="600"/>

   1）供给曲线表示的是在其他要不变的条件下，商品价格与供给量之间的关系。  
   2）不同的商品有不同的供给曲线，各种产品因扩大规模的难易程度不同，其供给曲线也各有差别。

5. 供给量的变动与供给的变动

   供给量的变动是指在某个时期内，当其他条件不变的情况下，由于商品价格变动所引起的企业愿意和能够提供的商品数量发生变动。

   供给的变动是指在某商品价格不变的条件下，由于其他因素变动所引起的该商品的供给数量的变动。

   <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-6.png" width="600"/>

## 2.3 市场均衡分析

1. 市场均衡

   所谓市场均衡，是指在影响需求和供给的其他因素都给定不变的条件下，市场上的商品价格达到这样一种水平，即使得消费者愿意购买的数量等于生产者愿意供给的数量。

   <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-7.png" width="600"/>

   用函数形式将市场均衡状态表示为: **Qd = Qs**

2. 市场均衡的变动

   需求的变动与市场均衡量的变动

   <image src="https://szu-csse-1305214533.cos.ap-guangzhou.myqcloud.com/economy%2F2-8.png" width="600"/>

   供给的变动与市场均衡量的变动  
   除了商品本身的价格以外，其他因素的变化都会引起供给曲线的移动。如随着生产技术水平的提高，在其他因素不变的情况下相同的价格下供给增加，供给曲线右移。

3. 价格管制对市场均衡的影响

   1）政府规定最低价格

   定义：是政府为了扶持某一行业的生产、保护生产者的利益而规定的高于该行业产品均衡价格的一种非市场价格。  
   支持价格的结果：供过于求，产品剩余；剩余产品由政府收购

   2）政府规定最高价格

   定义：是指政府为了防止某些生活必需品的物价上涨而规定了低于这些物品的均衡价格的价格，即规定最高价格。  
   限制价格的结果：供不应求，产品短缺；政府不得不实行配给制；出现黑市交易
