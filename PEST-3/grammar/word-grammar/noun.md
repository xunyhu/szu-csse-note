## 名词变化形式

**形容词变名词**

| 形容词             | 名词                  |
| ------------------ | --------------------- |
| safe（安全的）     | safety（安全）        |
| possible（可能的） | possibility（可能性） |
| existent（存在的） | existence（存在）     |
| popular 流行的     | popularity 流行       |

**动词变名词**

| 动词词        | 名词              |
| ------------- | ----------------- |
| exist（存在） | existence（存在） |
