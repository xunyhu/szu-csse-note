# 国际音标

## 前元音

- [ɪ]

  - 发音时嘴唇微张，口型扁平，舌尖抵下齿，舌头往前推，振动声带，发音短促。发音类似汉语“衣”。
  - 常见单词
    - **a**
      - orange /ˈɒrɪndʒ/
      - village /ˈvɪlɪdʒ/
    - **e**
      - delicious /dɪˈlɪʃəs/
      - telephone /ˈtelɪfəʊn/
    - **i**
      - sister /ˈsɪstə(r)/
      - ship /ʃɪp/
    - **u**
      - busy /ˈbɪzi/
      - minute /ˈmɪnɪt/
    - **y**
      - gym /dʒɪm/
      - happy /ˈhæpi/
    - **ey**
      - monkey /ˈmʌŋki/
      - money /ˈmʌni/
    - **ui**
      - build /bɪld/
      - guitar /ɡɪˈtɑː(r)/

- [i:]

  - 发音时嘴唇微张并尽量拉宽，舌尖抵下齿，舌前部自然拱起（略抬），振动声带，发音要延长。发音类似汉语“医”。
  - 常见单词

    - **e**
      - these /ðiːz/
      - evening /ˈiːvnɪŋ/
    - **i**
      - police /pəˈliːs/
      - magazine /ˌmæɡəˈziːn/
    - **ea**
      - tea /tiː/
      - seat /siːt/
    - **ee**
      - tree /triː/
      - beef /biːf/
    - **ei**
      - ceiling /ˈsiːlɪŋ/
      - receive /rɪˈsiːv/
    - **ey**
      - key /kiː/
      - keyboard /ˈkiːbɔːd/
    - **ie**
      - field /fiːld/
      - thief /θiːf/

  - [ɪ]&[i:]单词对比

    - ship /ʃɪp/; sheep /ʃiːp/;
    - miss /mɪs/; meet /miːt/;
    - sit /sɪt/; seat /siːt/;
    - live /lɪv/; leave /liːv/;
